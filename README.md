# Rota-Inteligente-Otimiza-o-de-Entregas-com-Algoritmos-de-IA
Projeto acadêmico que implementa uma solução de otimização de rotas de delivery usando algoritmos de Inteligência Artificial. Combina algoritmos de busca (A*, BFS, DFS) e aprendizado não supervisionado (K-Means) para criar rotas eficientes, reduzir o tempo de entrega e otimizar o consumo de combustível.
